Nombres Enters i Decimals

Nombres positius, negatius, decimals i arrodoniment

1 Nombres enters

Els nombres enters inclouen els positius, els negatius i el zero: ..., -3, -2, -1, 0, 1, 2, 3, ...

Z = { ..., -3, -2, -1, 0, +1, +2, +3, ... }

Regles d'operacions amb enters:

  • Suma de signes iguals: Es sumen els valors absoluts i es manté el signe. (+3) + (+5) = +8
  • Suma de signes diferents: Es resten els valors absoluts i es posa el signe del major. (+7) + (-3) = +4
  • Multiplicacio/Divisio: Mateixos signes = positiu, signes diferents = negatiu.
Exemples: (-4) + (-6) = -10 | (-3) × (-5) = +15 | (-12) ÷ (+4) = -3

2 Nombres decimals

Els nombres decimals tenen una part entera i una part decimal separades per una coma (o punt).

3,75 = 3 + 7/10 + 5/100

Tipus: exactes (0,5), periodics purs (0,333...), periodics mixtos (0,1666...).

3 Arrodoniment

Per arrodonir un nombre decimal a n xifres decimals:

  • Si la xifra seguent es 5 o mes: s'arrodoneix cap amunt.
  • Si la xifra seguent es menor que 5: es manté.
Exemples: 3,746 arrodonit a 1 decimal = 3,7 | 3,746 arrodonit a 2 decimals = 3,75
